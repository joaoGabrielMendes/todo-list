## Installation

1. Clone the repository
2. Create a local database, configure according to the file /server/config/db.config.js
3. In the root directory, run:

```
cd .\server\
npm install
npm start
```

Once the server is running, start the client:

```
cd .\client\
npm install
npm start
```
